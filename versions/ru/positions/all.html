Главная Позиции

Junior Java Developer


Разработчик Java - это специализированный программист, который сотрудничает с инженерами-программистами и веб-разработчиками для интеграции Java в программное обеспечение для бизнеса, приложения и веб-сайты.
Разработчик Java несет ответственность за выполнение множества обязанностей на протяжении всего процесса разработки цикла приложений. Это начинается от концепции к дизайну и тестированию.


Junior iOS Developer


Разработчик iOS отвечает за разработку приложений для мобильных устройств, работает на операционной системе Apple iOS. В идеале хороший iOS-разработчик владеет одним из двух языков программирования для этой платформы: Objective-C или Swift.


Junior Android Developer


Разработчик Android - это разработчик программного обеспечения, который специализируется на разработке приложения для Android Marketplace. Торговая площадка Android - это прямой конкурент Apple Store. Это означает, что большая часть работы разработчика Android связана с созданием приложений, которые мы используем на наших смартфонах и планшетах.


Junior QA Engineer


Инженер по обеспечению качества (qa) программного обеспечения - это тот, кто контролирует каждый этап процесса разработки программного обеспечения, чтобы обеспечить качество дизайна, убедившись, что ПО соответствует стандартам, установленным компанией-разработчиком. Инженеры по обеспечению качества программного обеспечения проверяют, чтобы новые продукты работали до их публикации.


Junior Motion Designer


Дизайнеры моушн-графики, которых иногда называют моушн-дизайнерами, создают произведение искусства для сайтов,телевидения или фильмов. Это могут быть видеоклипы, трейлеры, рекламные ролики и т.д. Они используют визуальные эффекты, анимацию и другие кинематографические приемы, чтобы дать жизнь их творениям.


Junior Frontend Developer


Фронтэнд разработчик проектирует и разрабатывает веб-сайты и приложения с использованием Интернета с помощью такаих технологий как HTML, CSS, DOM и JavaScript, которые работают в Open Web Platform или выступать в качестве входных данных для компиляции в средах разработки, не относящихся к веб-платформе (т.е. React Native).
