Додому Позиції

Junior Java Developer

Розробник Java - це спеціалізований програміст, який співпрацює з іншими інженерами програмного забезпечення та веб-розробниками для інтеграції Java у ділове програмне забезпечення, додатки та веб-сайти. Найчастіше Java розробники відповідають за Back-end розробку Розробник Java має багато обов’язків впродовж всього циклу розробки програмного забезпечення. Ці обов1язки можуть включати: створення концепції майбутнього застосунку, розробку конструкцій, а також тестування ПЗ.

Junior iOS Developer

iOS програміст відповідає за розробку додатків для мобільних пристроїв на базі операційної системи iOS від Apple. Хороший розробник iOS має володіти однією з двох найпопулярніших мов програмування для цієї платформи: Objective-C або Swift.

Junior Android Developer

Android програміст - це розробник програмного забезпечення, який спеціалізується на розробці додатків для платформи Android. Ринок Android є прямим конкурентом гіганта у сфері програмного забезпечення — Apple. Більшість завдань розробника Android зводиться до створення програм, які ми використовуємо на своїх смартфонах і планшетах.

Junior QA Engineer

Інженер із забезпечення якості програмного забезпечення (Quality Assurance Engineer) - людина, яка стежить за кожним етапом процесу розробки програмного забезпечення. Саме вона відповідальна за якість продукту на кожному етапі розробки. QA інженер перевіряє чи відповідає програмне забезпечення стандартам, встановленим розробницькою компанією. Інженери із забезпечення якості програмного забезпечення перевіряють працеспроможність нових продукти ще до їх випуску в широкий загал.

Junior Motion Designer

Motion дизайнери або ж дизайнери що відповідають за рух графіки, митці в широкому сенсі цього слова — створюють рухому графіку та анімацію для Інтернету, телебачення або кіно. Це може включати кліпи, трейлери, рекламні ролики, спливаючі послідовності баннерів тощо. Вони використовують візуальні ефекти, анімацію та інші кінематографічні прийоми, щоб оживити свої творіння.

Junior Frontend Developer

Фронт-енд розробник проектує та розробляє веб-сайти та додатки з використанням веб-технологій (HTML, CSS, DOM та JavaScript). Саме вони відповідають за функціональну складову та візуальне наповнення веб-сайтів.
