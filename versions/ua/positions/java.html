Junior Java Developer

Додому Позиції Junior Java Developer

Java Developer – це фахівець, основним завданням якого є створення складних корпоративних систем для обробки даних. До таких систем можна віднести настільні додатки, веб-сервіси, хмарні сховища даних, веб-додатки, а також складні банківські системи.

Java-програмісти зазвичай затребувані в тих областях, де використовують проекти з багатокористувацької обробкою даних і високими вимогами до рівня безпеки. Java-програміст може розробляти архітектуру програми, бізнес-логіку додатка, підтримує і супроводжує проект і пише технічну документацію.

Хто такий Java розробник?

Це розробник котрий створює різноманітне програмне забезпечення, використовуюючи мову програмування Java.

Що робить Java розробник?

За допомогою мови програмування Java та певних допоміжних технологій він створює:
- веб-застосунки (серверна частина);
- комп'ютерні ігри;
- додатки для робочого стола;
- мобільні застосунки під Android;
-хмарні середовища збереження даних;
- програмне забезпечення для банківських та корпоративних систем.

- підтримка і оновлення істуючих застосунків.

Вимоги до Java розробника:

- Глибокі знання мови програмування Java;
- Впевнені знання ООП та шаблонів проектування;
- Впевнені знання алгоритмів та структур даних;
- Знання баз даних (MySQL или PostgreSQL);
- Знання JDBC (API для взаємодії Java з базами даних) та Hibernate;
- Знання системи тестування застосунків (TDD);
- Знання мови розмітки XML та веб-технології JSON;
- Базові знання фреймворку Spring;
- Уміння аналізувати, освоювати і застосовувати нові технології;
- Англійська мова рівня не нижче Intermediate.
